# Gerald Ofori — Personal Portfolio 🚀

> A minimal, intentional, and high-performance personal portfolio showcasing digital frameworks, identity systems, and user interfaces designed with structural honesty.

## 📖 About The Project

This is the official personal portfolio of Gerald Ofori. Built with modern web technologies, the project emphasizes clean aesthetics, structured typography, and high performance. It features a curated list of selected professional projects spanning across public sector technology, fintech solutions, and corporate identity.

### ✨ Key Features

- **Modern Architecture**: Built with the latest Next.js App Router, React, and strict TypeScript.
- **Dynamic Previews**: Integrates the Microlink API to dynamically fetch and display live, up-to-date screenshots of deployed project URLs.
- **Responsive & Accessible**: Fully responsive layout designed with Tailwind CSS, ensuring a seamless and pixel-perfect experience across all devices.
- **Design System**: Implements a consistent, minimal design system with structural honesty and clean typography.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have the following installed on your machine:
- Node.js (v18.17 or higher)
- [pnpm](https://pnpm.io/) (recommended package manager)

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/geraldofori/portfoliio.git
   ```
2. Navigate into the project directory
   ```sh
   cd portfoliio
   ```
3. Install the project dependencies
   ```sh
   pnpm install
   ```
4. Start the development server
   ```sh
   pnpm run dev
   ```
5. Open your browser and navigate to [http://localhost:3000](http://localhost:3000) to view the application.

## 📁 Project Structure

- `app/` - Contains the Next.js App Router pages and layouts (`/about`, `/projects`, `/contact`).
- `components/` - Reusable UI components including the `Navbar` and `Footer`.
- `lib/` - Utility functions, application constants, and the mock database (`projects.ts`).
- `public/` - Static assets like local images and favicons.
